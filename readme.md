### Как запустить проект

Склонировать репозиторий
```
git clone https://github.com/LLIALLIJLblK/tron_microservice.git
```
API KEY
```
вставьте ваш api key в корень проекта в файл .env
```
Запустите проект
```bash
make all
```
Запустить тесты
```bash
make test
```
Посмотреть логи
```
make logs
```
Остановить проект
```bash
make down
```
Удалить проект
```bash
make delete
```
